Source hotkeys

  • Ctrl + C Center source horizontal and vertical
  • Ctrl + Alt + C Center source horizontally
  • Ctrl + Shift + C Center source vertically
  • Ctrl + Alt + Left Move source to left of canvas
  • Ctrl + Alt + Right Move source to right of canvas
  • Ctrl + Alt + Up Move source to top of canvas
  • Ctrl + Alt + Down Move source to bottom of canvas
  • Ctrl + Up Move source up in order
  • Ctrl + Down Move source down in order
  • Ctrl + Home Move source to Top of list
  • Ctrl + End Move source to Bottom of list
  • Ctrl+F Fit source to screen
  • Ctrl+R Reset size of Source
  • Del Remove Source
  • Arrowkeys Move source around (pixel per pixel)

Modifier Keys

  • Ctrl Disable Source snapping to edges
  • Shift Ignore the Aspect Ratio while resizing a source
