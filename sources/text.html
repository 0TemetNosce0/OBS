Text

Add some Text to your Scene.

Settings

Font

Select your desired Font.

Font Size/Bold/Italic/Underline/Vertical

Select your desired Size and Font facing.

Color

Select the Text color.

Opacity

Select the desired opacity for your Text.

Scroll Speed

Activate a scrolling text, by changing the speed from 0 to your liking.

Background Color

Select a background color. (Only visible if Opacity is not 0)

Background Opacity

Select the opacity of the Text Background.

Use Text From File

Select a file to use its contents as Text.

Enter Text

Simply enter the Text you want on stream here.

Use Outline

Color

Select the color of your Outline.

Thickness

Choose the thickness for your Outline.

Opacity

Change the visibility of your Outline.

Use Custom Text Extents

Forces the text box to be a specific size.

Size

Sets the size of the text box.

Wrap

Forces the text to wrap and allows you to specify an alignment.

Align

Align the text to left, center or right.
