Sources

Sources can be added to every scene. Though if you want a persistent source, you can also add it as a Global Source.
  • Which capture to use?
    • Window Capture: Capture a single Window
    • Monitor Capture: Capture your whole Monitor
    • Game Capture: Capture a fullscreen Game
  • Video Capture Device: Capture a Webcam or Capture Card
  • Image: Add an Image to your scene
  • Image Slide Show: Add an Image Slide Show to your scene
  • Text: Add some Text to your Scene
  • Sub-Regions: Select a small section of your Window/Monitor capture
  • Hotkeys: List of available Hotkeys
  • Global Sources: Explanation of Global Sources
  • Plugins: How to use Plugins
